About Us

Virtual college Space help students to stay updated about what's going on in their college allows them to take part in various activities with the help of following features.

Account creation

System has feature for students to create account so that they can keep track of all the events and clubs they are part of

Club creation

Students can start there own clubs with the help of this features and conduct various club activities.

Event Creation

Event creation is the feature provided to club admins so that they can post events that are condected by their clubs and allow students to register for them
